Project Description

SmartTrack is a simple web-based expense manager built using HTML, CSS, and JavaScript. It allows users to manage multiple profiles, record their income, and track expenses under different categories. Each user’s data is saved securely in the browser using Local Storage, ensuring that information remains even after the page is refreshed. The goal of this project is to demonstrate how front-end web technologies can be used to create interactive and user-friendly applications without the need for any server or database.
